# KORA · prototipo app demo


## Entregables incluidos

- `index.html` · demo principal con vista celular, tablet y computador.
- `login.html` · acceso alterno por perfiles demo.
- `qr-checkin.html` · página de check-in QR para el pasaporte cultural.
- `assets/css/styles.css` · estilos globales.
- `assets/js/data.js` · datos de perfiles, artistas, eventos, playlists y misiones.
- `assets/js/app.js` · lógica de navegación, audio, guardados, gamificación, ajustes y login/logout.
- `assets/img/` · logos, avatares SVG, portadas, posters y QR.
- `assets/audio/` · 6 cápsulas de audio WAV originales para la demo.

## Funcionalidades principales

- Login/logout simulado con múltiples perfiles.
- Reproducción real de cápsulas de audio.
- Tablero de hallazgos con guardados.
- Feed de descubrimiento con contexto cultural.
- Player con cola de artistas y modo 30s.
- Mapa de eventos con check-in QR y recompensas.
- Pasaporte cultural con sellos, XP y rachas.
- Playlist local colaborativa.
- Compartir cápsulas a canales demo.
- Ajustes funcionales y estado persistente con localStorage.
